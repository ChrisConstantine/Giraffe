##Giraffe Ray Tracer
[中文版 README](./README.md)


###Features
* diffuse reflection, mirror reflection
* refraction
* anti-aliasing
* global illumination
* Russian Roulette
* Procedural Texture
* Perlin Noise
* Depth of field
* Acceleration data structures
	- BVH
		+	AABB (Axis-Aligned Bounding Box)
		+	DOP	(Discrete Oriented Polytopes)
* Object
	- Triangle
	- Sphere
	- Quad
