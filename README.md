##Giraffe 光线追踪器
[English Version of README](./README.en.md)

###特点
* 漫反射, 镜面反射
* 折射
* 反走样
* 全局光照
* Russian Roulette
* Procedural Texture
* Perlin Noise
* Depth of field
* 加速数据结构
	- BVH
		+	AABB (Axis-Aligned Bounding Box)
		+	DOP	(Discrete Oriented Polytopes)
* 物体
	- 三角形
	- 球
	- 四棱柱
	- 圆柱
