###光线追踪器
###基于 ["smallpt"](http://kevinbeason.com/smallpt/) 实现的蒙特卡洛算法光线追踪器
###Features:
* 漫反射
* 镜面反射
* 折射
* Motion Blur
* 反走样
* 全局渲染
* Russian Roulette
* BVH
	-	AABB (Axis-Aligned Bounding Box)
	-	DOP	(Discrete Oriented Polytopes)
