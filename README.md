###光线追踪器
###基于 [http://kevinbeason.com/smallpt/]("smallpt") 实现的蒙特卡洛算法光线追踪器
###Features:
* 漫反射
* 镜面反射
* 折射
* 全局渲染
* Russian Roulette
* 反走样
* BVH
	-	AABB (Axis-Aligned Bounding Box)
	-	DOP	(Discrete Oriented Polytopes)
