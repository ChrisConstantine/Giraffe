##光线追踪器
###特点
* 漫反射, 镜面反射
* 折射
* 反走样
* 全局渲染
* Russian Roulette
* Procedural Texture
* Perlin Noise
* Depth of field
* Bump Mapping
* 加速数据结构
	- BVH
		+	AABB (Axis-Aligned Bounding Box)
		+	DOP	(Discrete Oriented Polytopes)
